1. 제안 개요

1.1 제안 배경 및 목적

고객의 비즈니스 과제

슬리밍고는 15년간 축적된 자세/체형 관리 노하우와 전국 11개 지점이라는 강력한 자산을 보유하고 있습니다. 그러나 이 자산들이 하나로 연결되지 않아, 비즈니스 성장의 발목을 잡고 있습니다.

현재 과제비즈니스 영향해결 방안
지점별 고객 정보 분산 — 11개 지점이 각각 고객을 관리 중복 등록, 지점 이동 고객 이탈, 전사 현황 파악 불가 모든 지점 고객을 하나의 시스템에 통합
고객 여정 단절 — 상담, 프로그램, 측정, 제품 정보가 따로 관리됨 맞춤 케어 불가, 재구매 시점 놓침 전 과정을 타임라인으로 시각화
실시간 현황 파악 불가 — 본사에서 전 지점 현황을 즉시 확인할 수 없음 느린 의사결정, 지점 비교 불가 본사 실시간 대시보드 제공
고객 관리 사각지대 — 재방문 시기를 놓쳐 고객이 조용히 떠남 고객 유지율 저하 자동 리마인더로 고객 관리
상담 기록의 비효율 — 상담지를 수기로 옮겨 적거나, 음성 상담 내용을 일일이 타이핑 기록 누락, 입력 시간 낭비 AI가 이미지/음성을 텍스트로 자동 변환

프로젝트 목적

  1. 고객 정보 통합: 11개 지점의 신규 고객 데이터를 하나의 시스템으로 통합
  2. 고객 여정 시각화: 첫 방문 → 상담 → 프로그램 구매 → 재방문 → 제품 구매 → 지속 방문까지 전체 여정을 한눈에 보고 관리
  3. 운영 현황 파악: 본사에서 전 지점의 고객 유치, 매출, 고객 진행 현황을 실시간 확인
  4. 업무 효율화: 반복 업무를 시스템화하여 직원의 업무 부담 절감

기대 목표

목표기대 효과
고객 유지율 향상재방문율 20% 이상 향상
업무 시간 절감고객 정보 조회/입력 시간 50% 이상 단축
매출 가시성 확보본사에서 전 지점 실시간 조회 가능
고객 데이터 오류 감소통합 관리로 중복/누락 90% 이상 감소

구분포함 내용

참고: 시스템은 앞으로 오는 신규 고객부터 관리합니다. 기존 고객 데이터 일괄 이관은 포함하지 않습니다.

프로젝트 규모: 총 36개 기능, 약 12주(약 3개월) 개발 기간, 반응형 웹 (PC/태블릿/모바일)

1.3 핵심 가치 제안

"11개 지점의 고객을 하나로 연결하여, 첫 방문부터 지속 방문까지 전체 여정을 한눈에 관리하는 시스템을 구축합니다."

고객 여정 관리 — 핵심 기능

고객 한 명의 전체 여정을 한 화면의 타임라인에서 확인할 수 있습니다. 각 단계를 드래그하여 이동하거나 클릭하여 상세 내용을 바로 확인/수정할 수 있어, 직관적으로 고객 상태를 관리합니다.

지속 방문프로그램 진행, 재방문 알림 관리

기존 솔루션 비교

비교 항목전문 CRM (Salesforce 등)본 프로젝트
장점고도화된 고객 관리슬리밍고 맞춤 고객 여정 관리
측정 데이터추가 비용 필요기본 포함 (before/after 비교)

3. 주요 사용자 흐름

신규 고객 등록 → 측정 → 프로그램 배정

신규 고객 정보 입력 → 자세측정 결과 등록 → 프로그램 배정 → 구매 금액 기록 → 고객 여정 타임라인에서 전체 이력 확인

고객 검색 → 이전 이력 확인 → 이번 방문 측정 → before/after 변화 확인 → 상담/프로그램 기록

타 지점 고객 검색 → "끌어오기" 요청 → 담당 지점 변경 → 이전 이력 포함 전체 확인
